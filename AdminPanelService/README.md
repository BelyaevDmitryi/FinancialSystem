# Admin Panel Service

Сервис административной панели для управления и мониторинга системы.

## Порт
- **8008** - основной порт Admin Panel Service

## Функциональность

### Статистика системы

- GET `/admin/stats` - получить общую статистику системы
  - Общее количество пользователей
  - Общее количество ордеров
  - Количество активных ботов
  - Статистика ордеров по статусам
  - Статистика ботов по стратегиям

## Примеры использования

### Получение статистики
```bash
GET http://localhost:8008/admin/stats
```

## Health Check
```bash
GET http://localhost:8008/actuator/health
```
